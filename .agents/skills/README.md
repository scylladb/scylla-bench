# Agent Skills

This directory contains AI coding agent skills that provide specialized instructions and workflows for Go development tasks. These skills are used by OpenCode (and compatible tools) to give the AI agent domain-specific knowledge.

## Managing Skills

Skills are managed using the [`skills`](https://skills.sh/) CLI tool (an npm package).

### Prerequisites

- Node.js (for npx)

### Adding a New Skill

Browse available skills at [skills.sh](https://skills.sh/), then install with:

```sh
npx skills add <skill-name>
```

For example:

```sh
npx skills add golang-testing
```

### Updating Skills

To update all installed skills to their latest versions:

```sh
npx skills update
```

To update a specific skill:

```sh
npx skills update <skill-name>
```

### Removing a Skill

```sh
npx skills remove <skill-name>
```

## Lock File

The `skills-lock.json` file in the repository root tracks installed skills, their sources, and content hashes. This file is auto-generated by the `skills` CLI and should be committed to version control to ensure reproducible setups across the team.
